**华为2016上机笔试题**

# 1. 最高分是多少？
老师想知道从某某同学当中，分数最高的是多少，现在请你编程模拟老师的询问。当然，老师有时候需要更新某位同学的成绩。

### 基本逻辑
按照题目的逻辑书写代码即可。
本体存在两个坑：1. 存在多组 nm; 2. 存在查询时输入 A > B 的情况

实现代码：
```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        while (true) {
            String line = br.readLine();
            if (line == null) break;
            String[] str1 = line.trim().split(" ");
            int n = Integer.parseInt(str1[0]);
            int t = Integer.parseInt(str1[1]);
            int[] grades = new int[n];
            line = br.readLine();
            String[] str2 = line.trim().split(" ");
            for (int i = 0; i < n; i++) 
                grades[i] = Integer.parseInt(str2[i]);
            for (int j = 0; j < t; j++) {
                line = br.readLine();
                String[] temp = line.trim().split(" ");
                int a = Integer.parseInt(temp[1]);
                int b = Integer.parseInt(temp[2]);
                if (temp[0].charAt(0) == 'Q')
                    if (a >= b)
                        System.out.println(getMax(b, a, grades));
                    else 
                        System.out.println(getMax(a, b, grades));
                else 
                    grades[a - 1] = b;
            }
        }
    }
    public static int getMax(int start, int end, int[] array) {
        int max = 0;
        for (int i = start - 1; i < end; i++)
            max = array[i] > max ? array[i] : max;
        return max;
    }
}
```
